A C#/WPF-based Survivalcraft game launcher with multi-version management, multi-threaded downloads, auto-extraction, and multi-language support.

## Features

### Core Functions
- **Multi-version Management** - Add, edit, delete multiple Survivalcraft versions (Net/Multiplayer, API/Modded)
- **Multi-threaded Download** - Resume, pause/resume, auto-fallback to single thread
- **Built-in Extraction** - ZIP/7z auto-extract with 7z.dll integration
- **Version Detection** - 7-level fallback for automatic latest version info

### UI & Experience
- **Dark Theme**
- **Right-click Menu** - Edit, delete, open folder for game list items
- **Download Task Panel** - Progress, speed, percentage, pause/stop/open folder
- **Language Selection** - First-launch overlay picker, title bar 🌐 button, supports Chinese/English/Russian
- **Welcome Hint** - Shows game adding guide when no game is selected

### Extra Features
- **Useful Links** - Official site, community, mod sites, version collection, external resources
- **About Dialog** - Version info, author, MIT License, QQ group (Chinese only, click copy / button to join)
- **Easter Egg** - Click 5 times in About dialog [laugh]
- **Custom Icon** - Wooden block icon (EXE icon + window icon)

## Configuration Files
- `apps.json` - Game list + download save path
- `lang.txt` - Current language preference (zh/en/ru)
- `skip_lang.txt` - Skip first-launch language selection

## Dependencies
- .NET 8.0 (Windows Desktop / WPF)
- [SevenZipExtractor](https://www.nuget.org/packages/SevenZipExtractor/) v1.0.19

## Build
```bash
dotnet build -c Release
```
Output: `bin\Release\net8.0-windows\SC Launch.exe`

### Create Installer
```powershell
# Generate ZIP portable package
.\build.bat

# Or use PowerShell (requires Inno Setup 6)
.\build.ps1
```

### Release Versions
- **SC_Launch_windows.zip** - Portable version, extract and run
- **SC_Launch_windows_install.exe** - Installer, requires [Inno Setup 6](https://jrsoftware.org/isinfo.php)

**Note**: Close the running program before rebuilding, otherwise it will fail due to file lock.
