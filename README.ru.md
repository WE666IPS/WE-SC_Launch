Лаунчер для игры Survivalcraft на C#/WPF с поддержкой мультиверсий, много поточных загрузок, автоизвлечения и мультиязычности.

## Возможности

### Основные функции
- **Мультиверсии** - Добавление, редактирование, удаление нескольких версий Survivalcraft (Net/Мультиплеер, API/Моды)
- **Много поточная загрузка** - Восстановление, пауза/продолжение, авто откат к одному потоку
- **Встроенное извлечение** - Авто извлечение ZIP/7z с интеграцией 7z.dll
- **Определение версии** - 7-уровневый fallback для авто определения последней версии

### Интерфейс
- **Тёмная тема**
- **Контекстное меню** - Редактирование, удаление, открытие папки для элементов списка
- **Панель загрузок** - Прогресс, скорость, проценты, пауза/остановка/открытие папки
- **Выбор языка** - Оверлей при первом запуске, кнопка 🌐, поддержка китайского/английского/русского
- **Подсказка** - Показывает руководство по добавлению игры при пустом списке

### Дополнительно
- **Полезные ссылки** - Официальный сайт, сообщество, моды, версии, внешние ресурсы
- **О программе** - Информация, автор, MIT License, QQ группа
- **Пасхалка** - 5 кликов в окне "О программе" [смех]
- **Иконка** - Деревянный блок

## Конфигурация
- `apps.json` - Список игр + путь сохранения
- `lang.txt` - Текущий язык (zh/en/ru)
- `skip_lang.txt` - Пропустить выбор языка при первом запуске

## Зависимости
- .NET 8.0 (Windows Desktop / WPF)
- [SevenZipExtractor](https://www.nuget.org/packages/SevenZipExtractor/) v1.0.19

## Сборка
```bash
dotnet build -c Release
```
Вывод: `bin\Release\net8.0-windows\SC Launch.exe`

**Примечание**: Закройте программу перед повторной сборкой, иначе ошибка из-за занятого файла.
